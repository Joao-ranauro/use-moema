    2. Run the existing screenshot script to capture the local site:
       `node scripts/screenshot.mjs full`
       This saves `screenshots/full-page.png`.
    3. Also run in sections mode to get viewport-by-viewport captures:
       `node scripts/screenshot.mjs`
       This saves `screenshots/section-01.png`, `screenshots/section-02.png`, etc.
    4. Now capture the production site for comparison. Use Playwright directly (same approach as screenshot.mjs but pointing to https://usemoema.com.br):
       ```
       node -e "
       const { chromium } = require('playwright');
       (async () => {
         const browser = await chromium.launch();
         const page = await browser.newPage({ viewport: { width: 1440, height: 900 } });
         await page.goto('https://usemoema.com.br', { waitUntil: 'networkidle' });
         await page.waitForTimeout(3000);
         await page.screenshot({ path: 'screenshots/production-full.png', fullPage: true });
         console.log('Saved: screenshots/production-full.png');
         await browser.close();
       })();
       "
       ```
       NOTE: If Playwright is not installed globally or the ESM/CJS import fails, try with `import()` or install playwright browsers first with `npx playwright install chromium`. Adapt the script as needed -- the goal is to capture a full-page screenshot of production.
